# California Housing Prices

This repository contains the code and dataset for predicting housing prices in California. The dataset used is the California Housing dataset, which includes various features such as location, number of rooms, population, median income, and median house value.

## Dataset

The dataset used in this project is the California Housing dataset. It provides information on housing prices and related features for various districts in California. The dataset includes both numerical and categorical variables, making it suitable for regression analysis and machine learning tasks.

## Contents

- `data/`: This folder contains the dataset files.
- `notebooks/`: This folder contains Jupyter notebooks for data exploration, preprocessing, and modeling.
- `models/`: This folder contains trained machine learning models.
- `utils/`: This folder contains utility functions and scripts used in the project.

## Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/california-housing-prices.git
   ```

2. Install the required dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Explore the Jupyter notebooks in the `notebooks/` folder to understand the data and analysis.

4. Run the notebooks to preprocess the data, train models, and evaluate performance.

5. The trained models can be found in the `models/` folder.
